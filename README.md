# it-support-home-lab

This repository documents a hands-on IT support and helpdesk home lab built using Windows Server 2022 and Windows 10 virtual machines. Each lab simulates a real world enterprise support scenario that a helpdesk technician or IT support specialist would encounter on the job.

The goal of this lab is to build practical experience with the tools and tasks that come up daily in IT support roles including Active Directory, Group Policy, DNS, DHCP, remote support, ticketing systems, and Windows troubleshooting.

---

## Lab Environment

| Component | Platform | Role |
|---|---|---|
| Windows Server 2022 | VMware Workstation Pro | Domain Controller, DNS, DHCP |
| Windows 10 | VirtualBox | Domain joined client workstation |
| osTicket | Self hosted | Helpdesk ticketing system |
